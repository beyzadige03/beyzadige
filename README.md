# Prompt Eğitmeni (Öğrenci Arayüzü)

Bu proje, öğrencilerin yapay zekâya daha etkili prompt yazmayı öğrenmesi için hazırlanmış sade ve oyunlaştırılmış bir web arayüzüdür.

## Özellikler
- Prompt giriş alanı + yönlendirici mizahi metin
- Kategori seçimi (genel, hikaye, kod, açıklama, özet)
- "Analiz Et" butonu ile otomatik değerlendirme
- 100 üzerinden puanlama sistemi
- Eksik noktaları listeleyen geri bildirim paneli
- Daha iyi prompt için otomatik öneri üretimi
- İyi prompt örnekleri
- Eğlenceli imza/telif bölümü

## Çalıştırma
`index.html` dosyasını tarayıcıda açmanız yeterli.
