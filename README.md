# ÇERKEZKÖY BELEDİYESİ BOŞ PARK ALAN UYGULAMASI

Arduino maket mantığına uygun park simülasyonu:
- Yeşil ışık: Müsait park yeri
- Kırmızı ışık: Dolu park yeri
- Bazı alanlar sürekli kırmızı: Ev sahibi / engelli / dükkan sahibine özel
- Normal parkta araç en fazla 1 dakika kalır
- Son 10 saniyede uyarı: "Aracınızı çekin yoksa park cezası alacaksınız"

## Çalıştırma
`index.html` dosyasını tarayıcıda açın.
